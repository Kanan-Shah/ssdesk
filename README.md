SMART SUPPORT DESK - Ticketing, SLA & Analytics System

Overview:
- SSDESK is an end-to-end ticket management system that intelligently computes ticket prority, SLA deadlines, and provides analytics for support teams.

Tech Stack:
- Backend: Django + Django REST Framework
- Database: PostgreSQL
- Frontend: Django Templates + HTML/CSS + minimal JS

- Django was chosen for its rapid development capabilities, built-in ORM, and strong support for relational data modeling. 
- PostgreSQL was selected for its ability to handle structured and semi-structured data (JSON fields) efficiently, which is critical for storing ticket event metadata. 


Features:
- 1.Ticket Creation and Tracking
- 2.Priority Computation Engine
- 3.SLA Management
- 4.Event based history tracking 
- 5.Analytics Dashboard
